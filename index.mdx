Ferromax ERP is an integrated management platform built for Argentine hardware stores (*ferreterías*). It combines a Spring Boot REST API backend with a React frontend to deliver a point-of-sale terminal, real-time inventory tracking, automated invoice processing, and a consumer-facing online store — all in a single cohesive system.

## Get Started in 4 Steps

<Steps>
  <Step title="Set up PostgreSQL">
    Create the `ferromax_db` database and configure credentials in `src/main/resources/application.properties`.
  </Step>
  <Step title="Start the backend">
    Run `mvn spring-boot:run` from the project root. The API will be available at `http://localhost:8081/api`.
  </Step>
  <Step title="Start the frontend">
    Run `npm install && npm run dev` from the `ferromax-web/` directory. The app opens at `http://localhost:5173`.
  </Step>
  <Step title="Log in">
    Use the default admin credentials (`jose@ferromax.com` / `admin123`) to access the dashboard, or the employee credentials (`manuel@ferromax.com` / `emp123`) to open the POS terminal.
  </Step>
</Steps>

<Note>
  Product images (~213 MB) are not included in the repository. Copy the contents of `img.zip` (available from the development team) into `ferromax-web/public/img/` before starting the frontend.
</Note>
